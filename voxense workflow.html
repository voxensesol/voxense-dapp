VOXENSE · SPATIAL DEPIN WORKFLOW
From Node Sensing to Marketplace Tiles

A visual overview of how VOXENSE transforms real-world sensor activity into verifiable on-chain proofs, spatial data tiles, and marketplace assets that reward participating nodes.

STEP 1
Node Sensing Layer

Mobile or IoT devices act as VOXENSE nodes, capturing the physical world in real time.

  • GPS position
  • Motion (accelerometer & gyro)
  • Ambient (light, temp, humidity)
  • Audio fingerprints (patterns, not raw sound)
STEP 2
Local Privacy Engine

All processing happens on-device. Raw sensor data is never uploaded or stored on-chain.

  • Hash readings into fingerprints (SHA-256)
  • Sign payload with Ed25519
  • Build compact proof package
  • Discard raw sensor content
STEP 3
VOXENSE SDK Transport

The SDK sends the hashed and signed proof to the on-chain program over Solana RPC.

  • Python & TypeScript client APIs
  • Standardized proof payload format
  • Submit transactions to voxense-core
  • Listen for PoS verification events
STEP 4
On-Chain Proof-of-Sensing

The core VOXENSE program verifies that each proof is valid and consistent with reality.

  • Check node registration & ownership
  • Validate signature & structure
  • Cross-node spatial & temporal correlation
  • Accept or reject as Verified Reality
STEP 5
Spatial Ledger & Aggregation

Accepted proofs are written into the Spatial Ledger and grouped into spatial data tiles.

  • Store node, sensor type, hash, timestamp
  • Aggregate proofs by region & time window
  • Build GPS / motion / ambient / audio tiles
  • Prepare encrypted off-chain data bundles
STEP 6
Spatial Marketplace

Verified tiles become economic assets that can be discovered and purchased by buyers.

  • List encrypted spatial tiles
  • Show verification metadata
  • Price tiles in $VOX
  • Enable AI, robotics & research use cases
STEP 7
Node Rewards Flow

Node operators are rewarded when their validated sensing data contributes to sold tiles.

  • Buyer pays in $VOX
  • Marketplace fee is taken
  • Remaining value goes to reward pool
  • Contributors are paid based on reputation & contribution

Full loop: Node sensing → Local hashing → On-chain PoS → Spatial tiles → Marketplace → Node rewards.
